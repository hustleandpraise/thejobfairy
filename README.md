#thejobfairy

Search for jobs from twitter!

## Install & Run

```
npm install
```

The App can be run with 

```
node (or io) ./bin/www && ./bin/stream
``` 

or with nodemon 

```
node (or io) ./bin/www && ./bin/stream
```

Theres a nodemon config file included to ignore changes in the public and views folder.
